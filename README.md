# 🎓 AI-Based Student Performance Classification System

An AI-powered Machine Learning project that predicts and classifies student academic performance using supervised learning algorithms. The system analyzes student data such as study hours, attendance, assignments completed, and internal marks to predict:

* ✅ PASS / FAIL Status
* ✅ Performance Category (GOOD / AVERAGE / POOR)
* ✅ Grade Classification (A / B / C / D / F)

This project demonstrates the complete Machine Learning workflow including dataset generation, preprocessing, model training, evaluation, visualization, and real-time prediction.

# 📊 Input Features

The AI model uses the following student features:

| Feature              | Description                     |
| -------------------- | ------------------------------- |
| StudyHours           | Hours spent studying            |
| Attendance           | Attendance percentage           |
| AssignmentsCompleted | Number of assignments completed |
| InternalMarks        | Internal examination marks      |
